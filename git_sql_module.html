1. ¿Qué es Git?

**Git** es un **Sistema de Control de Versiones**. Piensa en él como una máquina del tiempo para tu código. Te permite guardar "instantáneas" (llamadas *commits*) de tu proyecto a medida que avanzas.

Si cometes un error, puedes "retroceder en el tiempo" a una versión anterior que funcionaba. También es esencial para **trabajar en equipo**, permitiendo que múltiples desarrolladores trabajen en el mismo proyecto sin sobrescribir el trabajo de los demás.

2. Comandos Básicos de Git

Trabajas con Git desde la terminal. El flujo básico es: Modificas archivos, los "preparas" (`add`), guardas la instantánea (`commit`) y la subes a un repositorio remoto como GitHub (`push`). 


# 1. Inicializa un repositorio de Git en tu carpeta
$ git init

# 2. Revisa el estado de tus archivos
$ git status

# 3. "Prepara" un archivo para guardarlo
$ git add index.html

# 4. "Guarda la instantánea" (commit) con un mensaje descriptivo
$ git commit -m "Añadido el index.html inicial"

# 5. Sube tus commits a un servidor remoto (como GitHub)
$ git push origin main

3. ¿Qué es SQL?

**SQL** (Structured Query Language) es el lenguaje estándar para comunicarse con **Bases de Datos Relacionales** (como MySQL, PostgreSQL, etc.).

Imagina una base de datos como una hoja de cálculo de Excel gigante y súper potente. SQL es el lenguaje que usas para pedirle datos (`SELECT`), añadir nuevas filas (`INSERT`), modificar filas existentes (`UPDATE`) o borrarlas (`DELETE`). Es la base de casi todo el backend.

4. Comandos Básicos de SQL (CRUD)

Las cuatro operaciones básicas se conocen como **CRUD**: Create (Crear), Read (Leer), Update (Actualizar) y Delete (Borrar). 


-- Esto es un comentario en SQL

-- READ (Leer): Selecciona datos de una tabla 'usuarios'
SELECT nombre, email FROM usuarios WHERE puntos > 100;

-- CREATE (Crear): Inserta una nueva fila en la tabla 'usuarios'
INSERT INTO usuarios (nombre, email, password)
VALUES ('Alex', 'alex@email.com', 'pass123');

-- UPDATE (Actualizar): Modifica una fila existente
UPDATE usuarios
SET puntos = 110
WHERE nombre = 'Alex';

-- DELETE (Borrar): Elimina una fila
DELETE FROM usuarios WHERE nombre = 'Alex';
