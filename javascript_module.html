1. ¿Qué es JavaScript?

JavaScript (JS) es el **lenguaje de programación** que da vida a la web. Si HTML es el esqueleto y CSS es la piel, JavaScript es el **cerebro y los músculos**.

Permite crear contenido dinámico, controlar multimedia, animar imágenes y, básicamente, hacer que la página web responda a las acciones del usuario (como clics, envíos de formularios, etc.) sin necesidad de recargar la página.

2. Variables (let y const)

En el JavaScript moderno (ES6+), usamos let y const para declarar variables. (Evitamos usar var).

  • let: Se usa para variables que **pueden cambiar** su valor.
  • const: Se usa para constantes, variables que **no cambiarán** su valor. Es la opción preferida por defecto.

// Esto es un comentario en JavaScript

// 'const' porque mi nombre no cambiará
const nombre = "Alex";

// 'let' porque la puntuación puede cambiar
let puntuacion = 100;

puntuacion = 110; // Esto es válido

// nombre = "Juan"; // Esto daría un error, porque 'nombre' es una constante

3. Funciones y Arrow Functions

Las funciones son bloques de código reutilizables. ES6 introdujo una sintaxis más corta y limpia llamada **Arrow Functions** (Funciones Flecha). 


// Función tradicional
function sumar(a, b) {
  return a + b;
}

// Misma función con sintaxis Arrow Function
const sumarArrow = (a, b) => {
  return a + b;
};

// Si solo tiene una línea, se puede acortar más
const sumarSimple = (a, b) => a + b;

// Llamando a la función
let resultado = sumarSimple(5, 3); // resultado es 8

4. Manipulación del DOM

El **DOM** (Document Object Model) es la representación de tu HTML que JavaScript puede entender y modificar. Así es como hacemos que las páginas sean interactivas.

Los dos métodos más comunes son seleccionar un elemento y escuchar un evento (como un clic).
