Desafío: API de Tareas (TODO)

🎯 Tu Misión

Tu objetivo es crear un servidor web mínimo con Flask que exponga una ruta de API en /tareas y devuelva una lista de tareas en formato JSON.

📋 Requisitos (Python/Flask)

En tu archivo `app.py`, debes:

  • Importar Flask y jsonify.
  • Crear una instancia de la aplicación Flask.
  • Definir una lista de tareas (diccionarios de Python).
  • Crear una ruta (endpoint) en @app.route("/tareas", methods=['GET']).
  • Dentro de la función de esa ruta, usar jsonify() para devolver la lista de tareas.

💡 Código de Ejemplo


from flask import Flask, jsonify

app = Flask(__name__)

# Datos de ejemplo
tareas = [
  { "id": 1, "titulo": "Aprender Python", "completada": True },
  { "id": 2, "titulo": "Crear API", "completada": False }
]

@app.route("/tareas", methods=['GET'])
def obtener_tareas():
  return jsonify(tareas)

# (Esto es para ejecutarlo localmente)
if __name__ == '__main__':
  app.run(debug=True)
