1. ¿Qué es CSS?

CSS son las siglas de **Cascading Style Sheets** (Hojas de Estilo en Cascada). Es el lenguaje que usamos para **dar estilo** a nuestras páginas HTML.

Si HTML es el esqueleto, CSS es la **piel, la ropa y el maquillaje**. Se encarga de los colores, las fuentes, los espacios, las animaciones y cómo se ve la página en diferentes dispositivos (diseño responsive).

2. Sintaxis y Selectores

Una regla de CSS se compone de un **selector** y un **bloque de declaración**. El selector "apunta" al elemento HTML que quieres estilizar, y el bloque de declaración contiene las propiedades y valores. 


/* Esto es un comentario en CSS */

/* El selector es 'p' (todos los párrafos) */
p {
  color: white;           /* Propiedad: color, Valor: white */
  font-size: 16px;        /* Propiedad: font-size, Valor: 16px */
}

/* Selector de Clase */
.mi-clase {
  background: #2C008D;
}

/* Selector de ID */
#mi-id-unico {
  border: 1px solid red;
}

3. El Modelo de Caja (Box Model)

Todo elemento en HTML es una caja rectangular. El modelo de caja describe cómo se calcula el espacio que ocupa esta caja. Consiste en:

  • Contenido (Content): El texto, la imagen, etc.
  • Padding (Relleno): Espacio transparente *dentro* del borde.
  • Border (Borde): El borde que rodea al padding y al contenido.
  • Margin (Margen): Espacio transparente *fuera* del borde, que separa esta caja de otras.

.mi-caja {
  width: 300px;
  
  margin: 10px;      /* 10px de margen en los 4 lados */
  padding: 20px;     /* 20px de relleno en los 4 lados */
  border: 2px solid #7744E7;
}

4. Introducción a Flexbox

Flexbox es un módulo de CSS que facilita enormemente la alineación y distribución de elementos en un contenedor. ¡Es la herramienta principal para crear diseños modernos!

Para usarlo, aplicas display: flex; al contenedor padre. Luego, puedes usar propiedades como justify-content (para alinear horizontalmente) y align-items (para alinear verticalmente). 


.contenedor-padre {
  display: flex;
  justify-content: space-between; /* Pone espacio entre los hijos */
  align-items: center;            /* Centra los hijos verticalmente */
  height: 200px;
  background: #1a1a1a;
}
