Desafío: Filtrar Usuarios

🎯 Tu Misión

Tu objetivo es escribir una consulta SQL para seleccionar los nombres de todos los usuarios que se registraron a partir del año 2025.

📄 Esquema de la Tabla

Imagina que tienes una tabla llamada usuarios con la siguiente estructura: 

Tabla: usuarios
----------------------------------------------
| id (INTEGER) | nombre (TEXT) | fecha_registro (DATE) |
----------------------------------------------
| 1            | Alex          | 2024-05-15            |
| 2            | Maria         | 2025-01-02            |
| 3            | David         | 2024-11-30            |
| 4            | Sofia         | 2025-03-22            |
----------------------------------------------

📋 Requisitos (SQL)

Tu consulta debe:

  • Seleccionar (`SELECT`) solo la columna nombre.
  • Apuntar (`FROM`) a la tabla usuarios.
  • Filtrar (`WHERE`) por la columna fecha_registro.
  • La condición debe ser que la fecha sea **mayor o igual** a '2025-01-01'.
