Desafío: Botón de Alternar

🎯 Tu Misión

Tu objetivo es escribir el código JavaScript para hacer que un botón muestre u oculte un párrafo cada vez que se le hace clic.

📄 HTML Base (No modificar)

Usa este HTML para tu solución:


<button id="toggleBoton">Mostrar / Ocultar</button>

<p id="textoSecreto">
  ¡Este es el texto que debe aparecer y desaparecer!
</p>

<!-- (Asegúrate de incluir tu script JS) -->
<script src="solucion.js"></script>

📋 Requisitos (JavaScript)

En tu archivo `solucion.js`, debes:

  • Seleccionar el botón (con id="toggleBoton").
  • Seleccionar el párrafo (con id="textoSecreto").
  • Añadir un `addEventListener` al botón para el evento `click`.
  • Dentro del evento, alternar (toggle) la clase hidden de Tailwind en el párrafo.
Pista: Puedes usar parrafo.classList.toggle('hidden');

Resultado Esperado

¡Este es el texto que debe aparecer y desaparecer!

Estado Inicial: Al hacer clic en el botón, el párrafo de arriba debe ocultarse. Al hacer clic de nuevo, debe reaparecer.
