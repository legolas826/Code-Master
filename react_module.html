1. ¿Qué es React?

React es una **biblioteca de JavaScript** (no un framework) creada por Facebook para construir interfaces de usuario (UIs) rápidas e interactivas.

Su principal fortaleza es que te permite construir tu UI a base de pequeñas piezas reutilizables llamadas **Componentes**. React se encarga de "reaccionar" a los cambios en tus datos y actualizar la pantalla de forma muy eficiente, sin tener que recargar toda la página.

2. Componentes y JSX

Los componentes son como funciones de JavaScript que devuelven HTML. Para facilitar esto, React usa **JSX** (JavaScript XML), que nos permite escribir una sintaxis similar al HTML dentro de nuestros archivos JavaScript. 


// Esto es un componente funcional de React
// Es una función que devuelve JSX

function Saludo() {
  return <h1>Hola, bienvenido a React!</h1>;
}

// Puedes usar variables dentro de JSX con llaves {}
function SaludoPersonal(props) {
  const nombre = "Alex";
  return <h1>Hola, {nombre}!</h1>;
  // 'props' se explica en la siguiente sección
}

3. Estado (State) y Props

Esta es la parte más importante de React.

  • Props (Propiedades): Son datos que se pasan *desde* un componente padre *hacia* un componente hijo. Son inmutables (no se pueden cambiar desde el hijo).
  • State (Estado): Es la memoria interna de un componente. Son datos que el componente *gestiona por sí mismo* y que **pueden cambiar**. Cuando el 'state' cambia, React vuelve a renderizar el componente.

4. Hooks (useState)

Los **Hooks** son funciones especiales de React que nos permiten "engancharnos" a las características de React (como el 'state') desde componentes funcionales.

El hook más común es useState. Nos permite añadir 'state' (memoria) a nuestros componentes. 


import React, { useState } from 'react';

// Un componente 'Contador'
function Contador() {
  // 1. Declaramos una variable de estado 'conteo'
  //    'setConteo' es la función que usamos para actualizarla
  const [conteo, setConteo] = useState(0); // 0 es el valor inicial

  return (
    <div>
      <p>Has hecho clic {conteo} veces</p>
      
      {/* 2. Cuando se hace clic, llamamos a 'setConteo' */}
      <button onClick={() => setConteo(conteo + 1)}>
        Haz Clic
      </button>
    </div>
  );
}
