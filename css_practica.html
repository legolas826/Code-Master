Desafío: Centrado Perfecto

🎯 Tu Misión

Usando **solo CSS Flexbox**, debes centrar la caja hija (clase `.caja-hija`) perfectamente en medio de la caja padre (clase `.contenedor-padre`).

📄 HTML Base (No modificar)

Usa este HTML para tu solución:


<div class="contenedor-padre">
  <div class="caja-hija">
    Centrado!
  </div>
</div>

📋 Requisitos (CSS)

Debes escribir el CSS para `.contenedor-padre`:

  • Debe usar display: flex;.
  • Debe usar justify-content para el centrado horizontal.
  • Debe usar align-items para el centrado vertical.
