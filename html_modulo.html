1. Introducción al HTML

HTML son las siglas de **HyperText Markup Language** (Lenguaje de Marcado de Hipertexto). No es un lenguaje de programación; es un **lenguaje de marcado** que le dice al navegador web cómo estructurar el contenido que ves.

Piensa en HTML como el **esqueleto** de una página web. Define los párrafos, los encabezados, las imágenes y los enlaces. Todo lo que ves en una página web comienza con HTML.

2. Estructura de un Documento

Todo documento HTML5 sigue una estructura básica. Comienza con la declaración <!DOCTYPE html>, seguida de la etiqueta <html>. Dentro, encontrarás el <head> (con metadatos y el título) y el <body> (con el contenido visible). 


<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Título de mi Página</title>
</head>
<body>

  <h1>Hola, Mundo</h1>
  <p>Este es mi primer párrafo.</p>

</body>
</html>

3. Etiquetas Básicas

Usamos etiquetas para definir elementos:

  • <h1> a <h6>: Para encabezados (títulos y subtítulos).
  • <p>: Para párrafos de texto.
  • <a>: Para enlaces (anclas).
  • <img>: Para imágenes.
  • <div>: Un contenedor genérico para agrupar elementos.

4. Atributos

Las etiquetas pueden tener atributos, que proveen información adicional. Se colocan en la etiqueta de apertura. 


<!-- El atributo 'href' define la URL del enlace -->
<a href="https://www.google.com">Ir a Google</a>

<!-- Los atributos 'src' (fuente) y 'alt' (texto alternativo) son esenciales para imágenes -->
<img src="logo.png" alt="El logo de mi sitio">

5. HTML Semántico

HTML5 introdujo etiquetas que describen su contenido. Usar <header>, <footer>, <nav>, <main>, <section>, y <article> es mejor que usar <div> para todo, ya que le da más significado a tu página (bueno para SEO y accesibilidad).
